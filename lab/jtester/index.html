Jtester

Java Performance Testing Library

  • Method Running Time Test (Single Source)
  • Class Running Time Test (Package)

If you are interested in this project. You can find Sources at Jtester Github Repository

How to Use

1. Import Jtester or Fork this repository

2. Create Annotation at Your Test Class or Method

@Source(main=false)
class TestCode{

    @Test
    public void testFirst(){
        //...
    }

    @Test
    public void testSecond(){
        //...
    }

}

Test Class need @Source Annotation and If you want to test each Methods, add @Test Annotation.

3. Run Jtester

Method Running Time (@Test method)

Jtester tester = new Jtester();
tester.setInputPath("txt/input.txt");
tester.runMethods(TestCodeFirst.class);
tester.runMethods(TestCodeSecond.class);

Class Running Time (main method)

Jtester tester = new Jtester();
tester.runPackage("com.github.daegwang.jtester.test");
  • If you want to Run each Methods, change @Source Annotation to @Source(main=false).
  • Each Class need implement main method
